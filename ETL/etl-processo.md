# Processo de ETL (Extração, Transformação e Carga)

## Extração
- Download do dataset no Kaggle.

## Transformação (Power Query / Power BI)
- Filtragem do período para 2015 a 2022.
- Remoção de colunas vazias (ex.: latitude, longitude, regional, delegacia, uop).
- Ajuste de tipos de dados (datas e números).
- Seleção das colunas relevantes para análise e dashboard.

## Carga
- Carregamento do conjunto tratado no Power BI para construção do dashboard.
